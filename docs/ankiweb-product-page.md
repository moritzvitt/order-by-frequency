# Order By Frequency

Learn the words that matter first. 📚

`Order By Frequency` helps you reorder an Anki deck so common vocabulary shows up before rare words. Instead of studying in import order, you can move through your deck in a way that feels much closer to real language use.

This is especially useful for language decks where some cards are clearly more worth learning early than others.

## Why People Use It

- See high-frequency words sooner
- Make large vocab decks feel less random
- Keep your study order aligned with real-world usage
- Preview everything before making changes
- Reuse your own local frequency lists

## What It Does

The add-on compares cards in your deck against one or more frequency lists, then:

- shows you a preview before anything is changed
- repositions new cards so more common words come first
- lets you choose which note fields should be checked first

## Good Fit For

- learners with a big vocabulary deck that needs better ordering
- people studying French with the bundled frequency data 🇫🇷
- anyone who already has a custom frequency list and wants to use it in Anki
- users who want more control without giving up a simple menu-driven workflow

## Why It Feels Safe

The add-on starts in preview mode by default. That means your first run shows you what would happen before anything is written. 👀

If the ranking looks right, you can switch off `dry_run` in the add-on config and apply it for real.

## Typical Workflow

1. Open `Tools -> Order Deck By Frequency`
2. Pick the deck you want to reorder
3. Review the ranked and unmatched preview
4. Apply the change when you're happy with it

## Included By Default

- a ready-to-use default setup for a deck named `French`
- bundled French frequency data
- support for multiple local frequency sources

## Simple, Practical, Focused

This add-on is not trying to turn Anki into a full language-learning platform.

It does one job: help your deck surface more useful vocabulary earlier, with a workflow that stays transparent and easy to check. ✨
