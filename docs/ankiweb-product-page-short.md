# Order By Frequency

Study the more useful words first. 📚

`Order By Frequency` compares the cards in your deck against a frequency list and moves common vocabulary ahead of rarer words.

It can:

- preview the ranking before making changes
- reorder new cards by frequency
- let you choose the language or source for each run

## How To Use It

1. Open `Tools -> Order Deck By Frequency`
2. Choose the deck
3. Choose the language or source
4. Review the preview
5. Apply the reorder if it looks right

## What It Is Good For

- large vocabulary decks that feel too random
- language study where common words should come earlier
- learners who want a safer workflow with a preview first
- people using bundled or custom frequency lists

The add-on starts in preview mode by default, so you can check the result before writing anything. 👀
